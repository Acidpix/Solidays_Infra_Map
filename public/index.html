Le flux live s'affiche via WebRTC (API Gateway). Le login est centralisé : le serveur s'authentifie avec un basic user partagé ; aucun identifiant n'est demandé à l'utilisateur. Chaque caméra remonte l'IP de son serveur Milestone via la macro Zabbix (multi-sites / fédéré géré automatiquement).
